## 🎨 Composition

<aside>
🔎

Most AI product shots are flat.

Product in the middle, background behind it, done.

Real editorial photography thinks in layers.

Something in front.

Something behind.

The product sharp in between.

One line adds it:

> “soft out-of-focus botanical elements in the foreground, blurred leaves and
flowers framing the edges, golden hour lighting, shallow depth of field”
> 

Here’s what each part does:

- → “Shallow depth of field” separates the three layers, foreground soft, product sharp, background soft
- → “Out-of-focus foreground” creates dimension, your eye knows something is closer than the product
- → “Golden hour lighting” low sun, warm tones, long shadows, a specific moment, not just “good light

Most prompts describe what’s in the shot.

This prompt describes what’s between you and the shot.

One layer is a photo.

Three layers is editorial.

</aside>

# Contexte projet

Tu vas m’aider à transformer le projet **AI Filmmaking / Creative Production Wiki** en site web Astro.

Le site est basé sur un projet existant appelé master-ai-filmmaking. Il doit devenir une documentation pédagogique, visuelle et évolutive autour de la création d’images, de personnages, d’univers et de séquences vidéo avec l’IA.

L’objectif n’est pas de créer une simple formation “prompts IA”, mais une ressource de référence structurée, à mi-chemin entre :

- une documentation pédagogique type Bootstrap Docs ;
- un wiki créatif ;
- une bibliothèque de prompts ;
- un laboratoire de production visuelle ;
- un portfolio d’études de cas.

Le site doit d’abord servir à l’autoformation et au partage de connaissances, avec une possible évolution future vers du e-learning ou des ressources premium.

# Positionnement éditorial

Le fil conducteur du site est :

> Voir → Composer → Générer → Stabiliser → Mettre en scène → Animer → Monter → Documenter
> 

Le site doit apprendre à :

1. construire une image ;
2. comprendre les bases photo / cinéma : cadre, composition, lumière, couleur, profondeur ;
3. adapter ces notions à la génération d’images IA ;
4. créer des personnages consistants ;
5. créer des objets, lieux, outfits, univers et moodboards ;
6. transformer des images en plans vidéo ;
7. documenter des workflows reproductibles ;
8. construire des projets complets comme des études de production.

L’outil principal de référence est **Adobe Firefly**, mais l’architecture doit rester ouverte à d’autres outils cohérents : Kling, Hailuo, Runway, Midjourney, Photoshop, Premiere, etc.

Le site doit être **tool-aware**, mais pas **tool-dependent**.

---

# Direction UX / UI

Le design doit être moderne, élégant, sombre, pédagogique et très lisible.

Référence d’intention :

- documentation claire type Bootstrap Docs ;
- esthétique studio / laboratoire créatif ;
- interface éditoriale visuelle ;
- cartes pédagogiques ;
- composants réutilisables ;
- code de prompts facile à copier ;
- navigation intuitive ;
- structure évolutive.

Le site doit éviter l’effet “template générique” ou “formation marketing IA”. Il doit donner une impression de ressource sérieuse, créative, documentée et personnelle.
