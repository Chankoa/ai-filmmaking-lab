# Bibliothèque de sources — Master AI Filmmaking

Ce dossier conserve les documents bruts qui alimentent le site. Il ne constitue pas le contenu publié : chaque source doit être synthétisée, recoupée avec le projet, adaptée au vocabulaire du parcours et reliée à une leçon ou à une ressource.

## Règles de contribution

1. Déposer une source dans la famille correspondant à son usage, pas à son format.
2. Conserver le fichier original. Une traduction ou une synthèse devient un contenu séparé dans `src/content` ou `src/data`.
3. Documenter l'auteur ou le compte, l'URL, la date de consultation et les droits lorsqu'ils sont connus.
4. Ne pas publier directement une capture d'écran tierce. Extraire une notion, la reformuler et produire un exemple propre au projet.
5. Relier chaque nouvelle notion à une étape du parcours : Voir → Composer → Générer → Stabiliser → Mettre en scène → Animer → Monter → Documenter.
6. Qualifier la source : `brute`, `à vérifier`, `synthétisée`, `intégrée` ou `archivée`.

## Nommage recommandé

Pour les prochains ajouts :

```text
AAAA-MM-JJ__auteur-ou-source__notion__numero.ext
```

Exemple :

```text
2026-07-12__mumasadiko__amorce-sonore__05.png
```

Les captures historiques conservent leur nom d'origine afin d'éviter de perdre la correspondance avec les exports et les dates de collecte.

## État du corpus

- 105 fichiers sources classés dans 7 familles.
- Forte couverture : composition, mouvements caméra, continuité personnage, storyboard et shot-list.
- Couverture moyenne : écriture de prompt, mise en scène, réalisme de peau.
- Couverture faible : montage, son, droits et provenance, tests comparatifs entre outils, accessibilité et évaluation pédagogique.
